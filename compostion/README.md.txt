Philips Hue control via Quartz Composer
============================

This is a Quartz Composer composition that allows control of the Philips Hue Lights

An asynchronous command line plugin is included in this project. Source code can be found here https://github.com/justaddmusic/AsyncCommandLinePlugin

Read http://www.developers.meethue.com/documentation/getting-started to get you started.
